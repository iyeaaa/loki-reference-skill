## 7. IP → 회사 식별이 어려운 이유

### 7.1 현재 상황

모든 방문자(39명)가 **ISP**(SK Broadband, Korea Telecom, LG DACOM 등)로 분류되었습니다.

이는 **정상적인 결과**입니다. 한국의 대부분 기업은 자체 ASN(Autonomous System Number)을 보유하지 않고, ISP로부터 IP를 할당받아 사용합니다.

### 7.2 미국/유럽 vs 한국의 차이

**미국/유럽 대기업:**
```
대기업 (Google, Microsoft 등)
    ↓
자체 ASN 보유 (AS15169 = Google)
    ↓
자체 IP 블록 할당받음 (ARIN/RIPE에서)
    ↓
IP 조회시 → "Google LLC" 반환
```

**한국 대부분 기업:**
```
한국 기업 (삼성전자 본사 제외 대부분)
    ↓
ISP와 기업 인터넷 계약
    ↓
ISP가 보유한 IP 대역에서 할당
    ↓
IP 조회시 → "SK Broadband" 반환
```

### 7.3 왜 한국 기업은 자체 ASN이 없나?

| 요인 | 설명 |
|------|------|
| **비용** | ASN 등록/유지 비용 (연간 수백만원) + 전용선 비용 |
| **필요성 부재** | 일반 기업은 ISP 회선으로 충분 |
| **기술 인력** | BGP 라우팅 관리 전문가 필요 |
| **한국 인터넷 환경** | ISP 품질이 좋아서 자체 구축 동기 낮음 |

### 7.4 자체 ASN을 가진 한국 기업 (극소수)

| 기업 | ASN | 이유 |
|------|-----|------|
| 삼성SDS | AS9619 | 글로벌 네트워크, 데이터센터 운영 |
| 네이버 | AS23576 | 대규모 서비스, CDN 필요 |
| 카카오 | AS38099 | 대규모 트래픽 처리 |
| LG CNS | AS9316 | IT 인프라 사업 |

### 7.5 한국 ISP 시장 구조

```
┌────────────────────────────────────────────────┐
│  KT (약 40%)  │  SK (약 30%)  │  LG (약 20%)   │
├────────────────────────────────────────────────┤
│         전국 광케이블 인프라 보유               │
│         기업용 전용회선 서비스 제공             │
│         IP 대역 직접 관리                      │
└────────────────────────────────────────────────┘
                    ↓
┌────────────────────────────────────────────────┐
│  일반 기업들                                   │
│  - ISP의 "기업 인터넷" 상품 사용               │
│  - 고정 IP 1~16개 할당받음                     │
│  - IP 소유권은 ISP에 있음                      │
└────────────────────────────────────────────────┘
```
