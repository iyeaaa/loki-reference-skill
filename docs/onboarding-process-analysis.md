## 1. What (무엇을)

### 프로세스 개요
사용자가 회사 정보를 입력하면 자동으로 바이어를 찾고, 맞춤 이메일을 생성하는 온보딩 자동화 프로세스

### 최종 산출물
| 항목 | 결과 |
|------|------|
| 리드 수 | 30명 |
| 이메일 프리뷰 | 90개 (30명 × 3스텝) |
| 이메일 템플릿 | 3개 (introduction, follow_up_1, follow_up_2) |
| 캠페인 | 3스텝 시퀀스 |
| 총 소요 시간 | **8분 33초** (513,547ms) |

---

## 2. Why (왜)

### 목적
- 사용자가 직접 바이어를 찾고 이메일을 작성하는 시간 절약
- AI 기반 자동화로 초기 영업 파이프라인 구축 지원
- 해외 바이어 발굴 및 초기 영업 커뮤니케이션 자동화

### 병목 원인 분석
| 병목 | 원인 | 영향 |
|------|------|------|
| Jina 타임아웃 60초 | 응답 없는 웹사이트 대기 | 배치당 최대 60초 지연 |
| 유효 이메일 비율 63% | 19/30만 유효 | Hunter.io Fallback 추가 실행 필요 |
| Description Enrichment | 웹사이트 크롤링 병렬 처리 | 5.5분 소요 |

## 6. How (어떻게)

### 프로세스 흐름도

```
┌─────────────────────────────────────────────────────────────────┐
│                    사용자: 회사 정보 입력                        │
│                    (industry, target, country)                  │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 1: Discovery (바이어 찾기)                        ~5분    │
├─────────────────────────────────────────────────────────────────┤
│  1. BigQuery에서 5개 데이터소스 검색 (B2B, Apollo, Fresh...)    │
│  2. 30개 리드 발견                                              │
│  3. Jina로 웹사이트 크롤링 (10개씩 배치, 60초 타임아웃)         │
│  4. Hunter.io로 이메일 검증                                     │
│  5. Gemini로 회사 설명 생성                                     │
│  6. 부족하면 Hunter.io Fallback 실행                            │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 2: Customer Group (그룹 생성)                     ~1초    │
├─────────────────────────────────────────────────────────────────┤
│  1. 새 고객 그룹 생성                                           │
│  2. 30개 리드를 그룹에 추가                                     │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 3: Templates (이메일 템플릿 생성)                ~1.5분   │
├─────────────────────────────────────────────────────────────────┤
│  1. GPT-5-mini로 3개 템플릿 생성                                │
│     - introduction (첫 번째 접촉)                               │
│     - follow_up_1 (첫 번째 후속)                                │
│     - follow_up_2 (두 번째 후속)                                │
│  2. 각 템플릿 15~47초 소요                                      │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 4: Sequence (캠페인 설정)                         ~1초    │
├─────────────────────────────────────────────────────────────────┤
│  1. 3스텝 시퀀스 생성                                           │
│  2. 발송 간격 설정 (즉시, +3일, +7일)                           │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 5: Previews (맞춤 이메일 프리뷰)                  ~10초   │
├─────────────────────────────────────────────────────────────────┤
│  1. 30명 × 3스텝 = 90개 이메일 프리뷰 생성                      │
│  2. 변수 치환 ({{company_name}}, {{담당자명}} 등)               │
│  3. 언어별 번역 (필요시)                                        │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│ Phase 6: Complete (완료 처리)                           ~1초    │
├─────────────────────────────────────────────────────────────────┤
│  1. 온보딩 진행 상태 업데이트                                   │
│  2. 완료 알림 이메일 발송                                       │
│  3. 사용자에게 SSE로 100% 진행률 전송                           │
└─────────────────────────────────────────────────────────────────┘
```
